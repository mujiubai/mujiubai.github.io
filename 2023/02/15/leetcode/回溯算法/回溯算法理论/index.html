回溯法解决的问题

回溯法，一般可以解决如下几种问题：

  • 组合问题：N个数里面按一定规则找出k个数的集合
  • 切割问题：一个字符串按一定规则有几种切割方式
  • 子集问题：一个N个数的集合里有多少符合条件的子集
  • 排列问题：N个数按一定规则全排列，有几种排列方式
  • 棋盘问题：N皇后，解数独等等

如何理解回溯法

回溯法解决的问题都可以抽象为树形结构

因为回溯法解决的都是在集合中递归查找子集，集合的大小就构成了树的宽度，递归的深度，都构成的树的深度。

递归就要有终止条件，所以必然是一棵高度有限的树（N叉树）。

回溯法模板

回溯三部曲

  • 回溯函数模板返回值以及参数

回溯算法中函数返回值一般为void。

再来看一下参数，因为回溯算法需要的参数可不像二叉树递归的时候那么容易一次性确定下来，所以一般是先写逻辑，然后需要什么参数，就填什么参数。

void backtracking(参数)
  • 回溯函数终止条件

什么时候达到了终止条件，树中就可以看出，一般来说搜到叶子节点了，也就找到了满足条件的一条答案，把这个答案存放起来，并结束本层递归。

if (终止条件) {
    存放结果;
    return;
}
  • 回溯搜索的遍历过程

回溯法一般是在集合中递归搜索，集合的大小构成了树的宽度，递归的深度构成的树的深度。

for (选择：本层集合中元素（树中节点孩子的数量就是集合的大小）) {
    处理节点;
    backtracking(路径，选择列表); // 递归
    回溯，撤销处理结果
}

回溯算法模板框架如下：

void backtracking(参数) {
    if (终止条件) {
        存放结果;
        return;
    }

    for (选择：本层集合中元素（树中节点孩子的数量就是集合的大小）) {
        处理节点;
        backtracking(路径，选择列表); // 递归
        回溯，撤销处理结果
    }
}

去重方法

去重可分为两种情况：

  • 相同值位于一处
  • 相同值分散在不同地方

相同值位于一处

去除重复值主要是去除树同层之间的重复，而树不同层之间的相同值无需去除。而要做到这一点，则每次回溯遍历时，如果当前值与前值相同，且此值又是此次遍历中第一次遇见（说明前一个值是在树中上一层），那么此值应该选，否则不选。代码如下：

初始点去重

//40. 组合总和 II
class Solution {
    vector<vector<int>> ans;
    vector<int> temp;
public:
    void backtrace(const vector<int>& candidates,int index,int target){
        if(target==0){
            ans.push_back(temp);
            return;
        }
        for(int i=index;i<candidates.size()&&candidates[i]<=target;++i){
            if(i>index&&candidates[i]==candidates[i-1]) continue;//去重代码
            temp.push_back(candidates[i]);
            backtrace(candidates,i+1,target-candidates[i]);
            temp.pop_back();
        }

    }
    vector<vector<int>> combinationSum2(vector<int>& candidates, int target) {
        sort(candidates.begin(),candidates.end());
        backtrace(candidates,0,target);
        return ans;
    }
};

上面方法是一种特殊情况：遍历起始点是从上一层后面一个节点开始，使得可以用是否是第一个节点来去重。如果，遍历起始点是从其他地方开始，上述方法就无效了。

比如每次遍历都是从0开始，此时需要使用一个数组来记录每个值是否使用。当遇见两个相同值，如果前一个值被使用（添加进路径容器），说明当前值是在前一个值得下层，当前值可以使用。否则，说明前一个值没在路径中，此时如果使用当前值会造成重复。

全局used数组去重（单个used数组去重）

//47. 全排列 II
class Solution {
    vector<vector<int>> ans;
    vector<int> temp;
    vector<int> used;
public:
    void backtrace(vector<int> &nums){
        if(temp.size()==nums.size()){
            ans.push_back(temp);
            return;
        }
        for(int i=0;i<nums.size();++i){
            if(used[i]==1) continue;
            if(i>0&&nums[i]==nums[i-1]&&used[i-1]==0) continue;//去重
            used[i]=1;
            temp.push_back(nums[i]);
            backtrace(nums);
            temp.pop_back();
            used[i]=0;
        }
    }
    vector<vector<int>> permuteUnique(vector<int>& nums) {
        sort(nums.begin(),nums.end());
        used=vector<int>(nums.size(),0);
        backtrace(nums);
        return ans;
    }
};

相同值分散在不同地方

当相同值分散在各处时，此时就需要每次遍历的时候去重了。使用数组记录每个数是否使用过，在同一个循环中如果当前值被使用过，那么应该跳过，否则需要选中。

局部used数组去重（多个used数组去重）

//491. 递增子序列
class Solution {
    vector<vector<int>> ans;
    vector<int> temp;
    unordered_map<int,int> appear;
public:
    void backtrace(const vector<int>& nums,int index){
        if(index>=nums.size()) return;
        int used[201]={0};
        for(int i=index;i<nums.size();++i){
            if(used[nums[i]+100]==1) continue;//去重
            used[nums[i]+100]=1;
            if(temp.size()>0&&nums[i]<temp.back()) continue;
            temp.push_back(nums[i]);
            if(temp.size()>=2) ans.push_back(temp);
            backtrace(nums,i+1);
            temp.pop_back();
        }
    }
    vector<vector<int>> findSubsequences(vector<int>& nums) {
        backtrace(nums,0);
        return ans;
    }
};

总体来看，都可以使用这种方法进行去重，但这种去重方法是最耗费空间的，每次遍历都需要申请一个数组来记录状态。
