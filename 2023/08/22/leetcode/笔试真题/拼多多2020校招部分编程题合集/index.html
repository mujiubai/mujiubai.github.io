1：多多的魔术盒子

多多鸡有N个魔术盒子（编号1～N），其中编号为i的盒子里有i个球。
多多鸡让皮皮虾每次选择一个数字X（1 <= X <= N），多多鸡就会把球数量大于等于X个的盒子里的球减少X个。
通过观察，皮皮虾已经掌握了其中的奥秘，并且发现只要通过一定的操作顺序，可以用最少的次数将所有盒子里的球变没。

那么请问聪明的你，是否已经知道了应该如何操作呢？

每次取中间大小的数值进行减少即可，相当于求logn的值

  • 时间复杂度：O()
  • 空间复杂度：O()
#include <iostream>
using namespace std;

void deal(int n){
    int ans=0;
    while(n>0){
        n/=2;
        ++ans;
    }
    cout<<ans<<endl;
}

int main() {
    int T;
    cin>>T;
    for(int i=0;i<T;++i){
        int n;
        cin>>n;
        deal(n);
    }
}
// 64 位输出请用 printf("%lld")

2：多多的排列函数

没看懂题意，解释如下：

根据题意，只有当依次递减或递增时，结果才会最小，因此只有当n%4为1或2时，结果为1，其余为0

而求最大值可以看成是一直n-1个数的最小值，那么再添加一个n则为最大值。

  • 时间复杂度：O()
  • 空间复杂度：O()
#include <iostream>
#include <vector>
using namespace std;

void deal(vector<vector<int>> &dp){    
    for(int i=1;i<dp.size();++i){
        if(i%4==1||i%4==2) dp[i][0]=1;
        else dp[i][0]=0;
        dp[i][1]=abs(i-dp[i-1][0]);
    }
}

int main() {
    int T;
    cin>>T;
    vector<vector<int>> dp(100000+1,vector<int>(2,0));
    deal(dp);
    for(int i=0;i<T;++i){
        int n;
        cin>>n;
        cout<<dp[n][0]<<" "<<dp[n][1]<<endl;
    }
}
// 64 位输出请用 printf("%lld")

3：多多的电子字典

多多鸡打算造一本自己的电子字典，里面的所有单词都只由a和b组成。
每个单词的组成里a的数量不能超过N个且b的数量不能超过M个。
多多鸡的幸运数字是K，它打算把所有满足条件的单词里的字典序第K小的单词找出来，作为字典的封面。

最开始自己做的时候使用的深搜来做，但效率较低。

此题是一个字典序问题，有点类似leetcode440；

使用二维数组dp[i] [j]表示i个a和j个b能表示多少个单词数。需要注意dp为 unsigned long long，否则超出范围。

由于一个位置要么为a要么为b，那么当i个a字符和j个b字符时，如果固定首位为a，那么有dp[i-1] [j]种，固定b则有dp[i] [j-1]种，因此dp[i] [j] =dp[i-1] [j] +dp [i] [j-1]+2;

和lc440思想类似，如果dp[n-1] [m]+1大于等于k，那么说明首位为a，–k, –n; 否则首位为b, k-=dp[n-1] [m] +2（多了一个1是因为本身占1），不断类似操作得到最终答案

  • 空间复杂度：O()

        vector<vector<unsigned long long>> dp(n+1,vector<unsigned long long>(m+1,0));//
        for(int i=0;i<=n;++i) dp[i][0]=i;
        for(int j=0;j<=m;++j) dp[0][j]=j;
        for(int i=1;i<=n;++i){
            for(int j=1;j<=m;++j){
                dp[i][j]=dp[i-1][j]+dp[i][j-1]+2;

4：骰子期望

扔n个骰子，第i个骰子有可能投掷出Xi种等概率的不同的结果，数字从1到Xi。所有骰子的结果的最大值将作为最终结果。求最终结果的期望

求每个数字能出现的次数，例如数字5能出现的次数等于所有骰子min(5, nums[i])的乘积减去所有1、2、3、4出现的次数。然后求概率即可。注意，使用printf会出现内存溢出等问题，使用的c++限制小数位数进行输出。

  • 时间复杂度：O()
  • 空间复杂度：O()
#include <cstdio>
#include <iostream>
#include <vector>
#include<iomanip>
using namespace std;

int main() {
    int n;
    while (cin >> n) { // 注意 while 处理多个 case
        vector<int> nums(n);
        int maxN;
        for(int i=0;i<n;++i){
            cin>>nums[i];
            maxN=max(maxN,nums[i]);
        }
        vector<long double> counts(maxN);
        long double sum=0;
        for(int i=1;i<=maxN;++i){
            long double cur=1;
            for(int j=0;j<nums.size();++j){
                cur*=min(i,nums[j]);
            }
            cur-=sum;
            counts[i-1]=cur;
            sum+=cur;
            // cout<<cur<<endl;
        }
        long double ans=0;
        for(int i=1;i<=maxN;++i){
            ans+=i*(counts[i-1]/sum);
        }
        cout<<setiosflags(ios::fixed)<<setprecision(2)<<ans<<endl;
        // printf("%.2lf\n",ans);
        // cout<<ans<<endl;
    }
}
// 64 位输出请用 printf("%lld")

5：二维表第k大数

在一块长为n，宽为m的场地上，有n✖️m个1✖️1的单元格。每个单元格上的数字就是按照从1到n和1到m中的数的乘积。给出一个查询的值k，求出按照这个方式列举的的数中第k大的值v。

开始使用的优先队列来求，但内存爆了。

和leetcode668类似，只是一个求第k大，一个求第k小，可以转换为求第n*m-k+1小的数。

#include <cmath>
#include <iostream>
#include <queue>
#include <vector>
using namespace std;

int main() {
    int n,m,k;
    while (cin >> n >> m >> k) { // 注意 while 处理多个 case
        //优先队列
        // if(n>m) swap(n,m);
        // priority_queue<long long,vector<long long>,greater<long long>> que;
        // int len=sqrt(4*k)+1;
        // int ie=max(m-len,0);
        // int je=max(n-len,0);
        // for(int i=m;i>=ie;--i){
        //     for(int j=n;j>=je;--j){
        //         que.push((long long)i*j);
        //         if(que.size()>k) que.pop();
        //     }
        // }
        // cout<<que.top()<<endl;

        //lc668
        long long left=1,right=(long long)n*m;
        k=right-k+1;
        while(left<right){
            int mid=left+(right-left)/2;
            int count=mid/n*n;// mid/n表示都小于等于mid的行数量，再乘以n表示总个数
            for(int i=mid/n+1;i<=m;++i){
                count+=min(mid/i,n);
            }
            if(count>=k){
                right=mid;
            }else{
                left=mid+1;
            }
        }
        cout<<left<<endl;
    }
}
// 64 位输出请用 printf("%lld")
